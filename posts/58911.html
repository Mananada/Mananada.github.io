2.1 Components组件

    @Component最常用的几个选项:

组件生命周期

  1. ngOnChanges：组件数据更新
  2. ngOnInit：初始化组件，只被调用一次
  3. ngDoCheck：检测数据变化
  4. ngAfterContentInit：内容初始化后，只被调用一次
  5. ngAfterContentChecked：检测内容变化
  6. ngAfterViewInit：视图初始化后，只被调用一次
  7. ngAfterViewChecked：检测视图变化
  8. ngOnDestory：组件销毁

组件交互

组件挂载

父传子@Input

Vue父组件直接在子组件上绑定数据，在子组件中用props来接收

Angular父组件在子组件上绑定属性，子组件使用input来接收（input需要导入）

子传父数据@Output

类似于Vue在子组件中触发$emit事件，然后在父组件中监听

angular中父组件给子组件传递一个事件，子组件通过@Output弹射触发事件

  1. 在子组件中导入output和EventEmitter
  2. 子组件使用output装饰器装饰EventEmitter的实例
  3. 子组件通过EventEmitter实例emmit数据，父组件监听实例获取数据

子传父@ViewChild()

通过ViewChild获取子组件实例，获取子组件数据

3.2 表单

表单 API 汇总

基础类

  • FormControl 实例用于追踪单个表单控件的值和验证状态。
  • FormGroup 用于追踪一个表单控件组的值和状态。
  • FormArray 用于追踪表单控件数组的值和状态。
  • FormBuilder 一个可注入的服务，提供一些用于提供创建控件实例的工厂方法

基础指令

  • FormControlDirective 把一个独立的 FormControl 实例绑定到表单控件元素。

  • FormControlName 把一个现有 FormGroup 中的 FormControl 实例根据名字绑定到表单控件元素。

  • FormGroupDirective 把一个现有的 FormGroup 实例绑定到 DOM 元素。

  • FormGroupName 把一个内嵌的 FormGroup 实例绑定到一个 DOM 元素。

  • FormArrayName 把一个内嵌的 FormArray 实例绑定到一个 DOM 元素。
